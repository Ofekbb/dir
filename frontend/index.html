מקורות סריקה

Telegram

ה-Bot Token וה-Chat ID מוגדרים כ-GitHub Actions Secrets בשם TELEGRAM_BOT_TOKEN ו-TELEGRAM_CHAT_ID.

  1. צור בוט חדש דרך @BotFather
  2. שמור את ה-token שתקבל
  3. שלח הודעה לבוט, ואז בקש getUpdates למצוא את ה-chat_id שלך
  4. הוסף את שניהם ב-GitHub → Settings → Secrets → Actions

תדירות סריקה

הסריקה רצה דרך GitHub Actions Cron כל ~10 דקות.

קובץ ההגדרות: config.yaml בשורש הפרויקט.
